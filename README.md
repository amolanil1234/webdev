# Summer Art Camp — One-Page Website

A warm, elegant landing page for a **Summer Art Camp for Kids (Ages 5–15)**. Built with React + Vite.

## Design

- **Feel:** Safe, joyful, creative; trustworthy for parents; inspiring for children; premium but warm.
- **Colors:** Soft pastels — peach, sage green, lavender, cream — with subtle artistic textures.
- **Typography:** Cormorant Garamond (headings) + Nunito (body).

## Sections

1. **Hero** — Headline, subheading, emotional copy, Enroll / Book Demo CTAs  
2. **About** — Camp benefits (creativity, focus, fine motor, confidence, expression, less screen time)  
3. **Art Forms** — Grid: Lippan, Texture, 3D, Boho, Doodle, Bottle, Fabric, Pot, Fluid Art  
4. **Why Parents Love** — Trust points (safe, small batches, materials, beginner-friendly, exhibition)  
5. **Camp Details** — Age groups, timings, duration, location, fees, early bird  
6. **Gallery** — Placeholder grid for photos  
7. **Testimonials** — Parent quotes with star ratings  
8. **Contact & Enrollment** — Phone, WhatsApp, form, social links  

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```
